# Project

This folder contains five subfolders, each with its own project, designed to showcase different aspects of web development and JavaScript programming.

### 📁[BasicJSProject](https://github.com/niveditakaur/Triweb-Learning/blob/main/Project/BasicJSProject/README.md)

- **Description**: This project allows users to enter text, pick a display position, and place it in one of four block spaces. This showcases how HTML, CSS, and JavaScript work together for an engaging and straightforward user interaction.

### 📁[HTML-CSS-JS-Project](https://github.com/niveditakaur/Triweb-Learning/blob/main/Project/HTML-CSS-JS-Project/README.md)

- **Description**: This project features a user registration form styled with CSS, structured in HTML, and enhanced by JavaScript for input validation, ensuring accurate and complete user information before submission.

### 📁[Portfolio](https://github.com/niveditakaur/Triweb-Learning/blob/main/Project/Portfolio/my-portfolio/README.md)

- **Description**: Created a React portfolio with parts for About, Skills, Experience, Projects, Education, and Contact, linked to Firebase. Added a navbar, animations, and used libraries like Styled Components, Toastify, Material-UI, Typewriter Effect, and React-Icons/FontAwesome for a lively and attractive user interface.

### 📁[QuizManager](https://github.com/niveditakaur/Triweb-Learning/tree/main/Project/QuizManager/README.md)

- **Description**: The Quiz Manager Backend, fueled by Node.js and MongoDB, enables easy quiz creation, publishing, and management while ensuring security with JWT-based authentication. Users can create, publish, and complete quizzes, generating detailed reports for analysis purposes.

### 📁[TicTacToe](https://github.com/niveditakaur/Triweb-Learning/tree/main/Project/TicTacToe/README.md)

- **Description**: This project is a Tic Tac Toe game created with React. It lets two players take turns on a 3x3 grid, displays the game status and winner.
